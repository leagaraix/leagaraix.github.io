Installation d'un poste de développement

Objectif : installation d’un système d’exploitation et des logiciels nécessaires à un environnement de travail, et rédaction d’un guide d’installation en anglais.

Compétences
  • Installer et configurer un système d’exploitation et des outils de développement
  • Rédiger un guide d’installation précis et clair
  • Maîtriser la langue anglaise dans un contexte professionnel

Nous avons effectué cet exercice deux fois pendant mon BUT Informatique. La première fois, nous avons installé Debian 11 avec interface graphique. Nous avions alors résumé les étapes d'installation sous forme d'une carte mentale. La seconde fois, nous avons installé Debian 12, mais en utilisant uniquement l'interface en lignes de commandes. Nous avons alors rédigé un guide complet, en anglais, pour détailler l'installation du système d'exploitation sur la machine virtuelle. Parmi les opérations réalisées, nous avons installé Apache2, PostgreSQL et PHP.

Mes réalisations

Installation de Debian 11 : avec KDE Plasma sous forme de machine virtuelle.

  • Installation d’un JDK : utilisation des packages Debian.
  • Installation d’un IDE.

Installation de Debian 12 : interface en lignes de commandes.

  • Installation de Apache2.
  • Installation de PostgreSQL : création d'une base de données, création et gestion d'utilisateurs, requêtes.
  • Rédaction d’un guide en parallèle de l’installation : distinction de chaque étape, repérage des sources d’erreurs, utilisation d'un vocabulaire précis en définissant les termes nouveaux.
