Création d’un site web à destination de collégiens

Objectif : réalisation d’un site web pour une ESN (Entreprise de Services du Numériques), à destination des collégiens et facilitant la demande de stages.

Compétences
  • Appréhender les besoins du client et de l’utilisateur
  • Maîtriser les langages de programmation HMTL et CSS
  • Concevoir une maquette de site web

Nous étions une équipe de 4. Après avoir rassemblé des informations sur l'entreprise cliente, nous avons conçu la maquette du site (un aperçu est disponible en bas de la page). Une fois que la maquette a été validée, nous avons commencé le développement. Ce travail s'est achevé avec une présentation orale du projet.

Réalisations de l'équipe

Recherches sur l’entreprise cliente : étude de ses caractéristiques, analyse de ses besoins. Nous avions pour modèle l'entreprise SCC.

Conception des pages : création de la maquette, ajustements aux demandes du client.

Développement :écriture du code HTML et CSS, corrections.

Adaptation du contenu du site au public (collégiens) : textes simples, peu de vocabulaire spécifique, page dédiée à la demande de stage, contact simplifié, sélection des informations utiles pour la compréhension du métier de l’entreprise.

Mes contributions

Recherches sur l'entreprise.

Conception de la maquette :

  • schéma sous forme d'une arborescence pour visualiser la hiérachisation du site,
  • vérification de la cohérence entre les pages,
  • ajout de commentaires pour faciliter la compréhension.

Apparence du site : rédaction de la majeure partie du code CSS.
